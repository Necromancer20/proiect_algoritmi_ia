## Manual Install

### Navigate to the Project Directory

### Set Up Virtual Environment

```bash
# On Windows
python -m venv env

# On macOS/Linux
python3 -m venv env
```

### Activate the Virtual Environment

```bash
# On Windows
.\env\Scripts\activate

# On macOS/Linux
source env/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the application

#### Windows

```batch
python menu.py
```

#### Linux/macOS

```bash
python3 menu.py
```
